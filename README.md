## 果冻调色约定

- 结构：`layer0` 为底图，`layer1` 为上层调色图。
- 运行时：`tintIndex=0` 不变色，`tintIndex=1` 只吃单个 ARGB tint 常量。
- 美术/提示词：可直接写“应用XYZ色彩空间原理”，含义是保持主色相稳定，用灰度明度组织通透感与层次。
- 推荐表达：
  > 对灰度图按不同明度进行分层上色，应用XYZ色彩空间原理，以指定颜色为基准色，保留亮度层次并保持整体通透的果冻质感。
